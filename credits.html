Web Hosting

Domain : Thanks to Mr. Bailey Cotton, who generously allowed us to host our website on a 1hsd.org domain for free. He communicated with us for any of our needs. Thank you, Mr. Cotton, for what you have done for us.

Web Hosting Service:  The web hosting provider where our code is stored is Github. Github has their own hosting service called Github Pages, where it is free to host educational and other related pages. If you would like to see the code running this page. Visit here , or click the Github icon in the footer. Github is also the CDN provider for hmsfbla.1hsd.org. 

SSL:  Let's Encrypt has partnered with Github to generate SSL Certificates for all Github Pages sites.

Website Builder:  While the majority of the code was created by Kelvin Orduna, some of the code and and 'blocks' were generated by Grapedrop . 
Code Editor:  Brackets  for Mac.
W3 Schools:  W3Schools also provided some of the code that was used in website.
